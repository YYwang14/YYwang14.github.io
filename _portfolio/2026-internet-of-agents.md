Developing an initial lab-scale Internet of Agents prototype for LLM agents, supporting agent publication and discovery through an ADS server based on Agntcy A2A/ADS concepts.

This work explores security challenges in open agent-to-agent ecosystems, including agent certification, privacy exposure, capability attestation, reputation, and trust establishment.
